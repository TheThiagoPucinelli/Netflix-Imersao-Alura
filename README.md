## 🎬 Netflix UI - Imersão Alura Front-End com IA!

### Resumo
A Imersão Front-end com IA da Alura é um evento gratuito e online que ensina os fundamentos do desenvolvimento web de forma prática.

Durante a imersão, os participantes aprendem HTML, CSS e JavaScript em poucas aulas, criando um projeto do zero com a orientação de especialistas. Além disso, o curso também introduz o uso de Inteligência Artificial como apoio no desenvolvimento, ajudando a escrever código, corrigir erros e acelerar o aprendizado.

A imersão conta com um cronograma de aulas, desafios práticos e uma comunidade ativa, onde os participantes podem tirar dúvidas, compartilhar projetos e evoluir juntos.

Projeto desenvolvido durante a Imersão Front-end com IA da Alura.

## 📌 Sobre

Interface inspirada na Netflix criada para praticar conceitos básicos de Front-end, utilizando HTML, CSS e JavaScript com apoio de ferramentas de IA.

## 🚀 Tecnologias

CSS
JavaScript

## 📚 Aprendizado

Projeto feito com foco em prática e aprendizado durante a imersão, explorando desenvolvimento de interfaces e uso de IA no código.
